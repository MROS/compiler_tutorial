有了遞迴與決策能力，音界咒已經能計算不少東西了。試寫了幾個有代表性又常見的術。

## 輾轉相除法

```音界
術．輾轉相除（甲、乙）【
    若（乙＝＝０）【
        歸．甲
    】
    歸．輾轉相除（乙、甲％乙）
】
```

## 冪次

```冪次
術．冪（數、次方）【
    若（次方＝＝０）【
        歸．１
    】
    歸．數＊冪（數、次方－１）
】
```

## 印出費氏數列前十項

```音界
術．費氏數（項）【
    若（項＜＝１）【
        歸．１
    】
    歸．費氏數（項－２）＋費氏數（項－１）
】

術．打印數列（項）【
    若（項＞１）【
        打印數列（項－１）
    】
    曰（費氏數（項））
】

術．初（）【
    打印數列（１０）
    歸．０
】
```
